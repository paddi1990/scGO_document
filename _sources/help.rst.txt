.. _help:

Getting Help
==================

We appreciate your feedback and questions! You can report any error or suggestion related to scGO as an `issue on github <https://github.com/yulab2021/scGO/issues>`_. If you have questions related to the manuscript, data, or any general comment or suggestion please use the `Discussions <https://github.com/yulab2021/scGO/discussions>`_.

Thank you!
